# Atomic Libs

Atomic Libs is a set of packages built and maintained by Atomic Jolt.

:::info Documentation
The documentation for these packages are still in-progress
:::

## Maintained Packages

This repo contains the following NPM packages:

- [`@atomicjolt/atomic-elements`](https://www.npmjs.com/package/@atomicjolt/atomic-elements): A library of reusable React components.
- [`@atomicjolt/atomic-fuel`](https://www.npmjs.com/package/@atomicjolt/atomic-fuel): Various utilties and helpers for building our applications.
- [`@atomicjolt/hooks`](https://www.npmjs.com/package/@atomicjolt/hooks): Custom React hooks for various use cases.
- [`@atomicjolt/forms`](https://www.npmjs.com/package/@atomicjolt/forms): Form handling utilities and components.
- [`@atomicjolt/lti-types`](https://www.npmjs.com/package/@atomicjolt/lti-types): TypeScript types for LTI applications.
- [`@atomicjolt/lti-client`](https://www.npmjs.com/package/@atomicjolt/lti-client): Client-side utilities for LTI applications.
- [`@atomicjolt/lti-server`](https://www.npmjs.com/package/@atomicjolt/lti-server): Server-side utilities for LTI applications.
- [`@atomicjolt/lti-components`](https://www.npmjs.com/package/@atomicjolt/lti-components): React components for LTI applications.
