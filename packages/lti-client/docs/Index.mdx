# Introduction

The `@atomicjolt/lti-client` package provides a streamlined interface for handling LTI (Learning Tools Interoperability) launches and communications in client-side JavaScript applications.

## Features

- Handle OIDC-compliant LTI 1.3 launch flows
- Support for deep linking launches
- Names and roles provisioning service integration
- Assignment and grades service support
- Post-message communication utilities
- Platform storage integration

## Installation

```bash
npm i @atomicjolt/lti-client
```

```bash
yarn add @atomicjolt/lti-client
```
